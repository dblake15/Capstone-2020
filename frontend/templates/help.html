How to use Launch - Quickstart Guide

You've gotten to the web-page, now what? This tutorial will walk you through the basics of how to use Launch.

  1. When you first navigate to Launch, you are met with a screen that looks like this:
  2. Enter in your GitHub username, or the username under which the project you are trying to deploy has been stored.
  3. Click the 'Begin' button, and the next page will auto-populate with your repositories.
  4. The next page should look like this:
  5. Select your project's repository from the drop-down, and make sure the 'Create' option is selected, then press the 'Go' button
  6. After a few seconds, you should see a message saying your application has been deployed
  7. Make a note of the port it is running on, and navigate to the IP address of your cluster, followed by the port you noted earlier.
  8. For example, if your project is running on port 32123, and your cluster IP address is 192.168.90.100, your project lives at http://192.168.90.100:32123
